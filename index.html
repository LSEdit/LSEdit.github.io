Abstract

Training-free image editing aims to modify images according to textual instructions without updating model parameters. Despite the strong generative priors offered by recent rectified-flow foundation models, preserving instruction-irrelevant content remains challenging when performing local edits. Existing methods often impose preservation constraints through global trajectories, spatial regions, or token partitions. Such coarse control can lead to an unfavorable trade-off in local editing, weak constraints may cause over-editing and source drift, whereas overly strong constraints may suppress desired semantic changes. In this work, we propose LSEdit, a training-free instruction-based image editing framework based on locally-standardized semantic-value modulation. LSEdit integrates instruction-related semantic priors and local-neighborhood-aware value deviations into continuous preservation weights. These weights directly modulate attention value representations, anchoring source-stable content while releasing edit-relevant semantic responses. To further stabilize ambiguous responses, we design a transition-band source stabilization mechanism that softly pulls uncertain value features toward the reference branch without over-constraining edit-relevant regions. Extensive experiments on public image editing benchmarks demonstrate that LSEdit substantially improves structure fidelity and background preservation across rectified-flow backbones.
